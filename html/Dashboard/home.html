    용어 설명

    • 작업일(Work Day)

      1) 주 : 일요일 ~ 토요일 기준으로, 토요일, 일요일, 휴일을 제외하고 카운트합니다.

      2) 월 : 매월 01일 ~ 말일 까지, 토요일, 일요일, 휴일을 제외하고 카운트합니다

    • 목표건수 : 각 과제별로 부서의 목표 건을 일일목표로 환산하여 관리자 메뉴의 "과제관리"화면에서 등록합니다.

      1) 일별 목표 건수 : 목표건수 (과제관리 화면에서 부서별로 등록한 값)

      2) 주별 목표 건수 : 일별목표건수 * 작업일

      3) 월별 목표 건수 : 일별목표건수 * 작업일

    • 절감시간 : 각 과제 1건이 수행될 때 절감되는 시간으로 관리자 메뉴의 "과제관리"화면에서 등록합니다.
    • 목표절감시간

      1) 일별 목표 절감시간 : 일별목표건수 * 건별절감시간 

      2) 주별 목표 절감시간 : 일별목표건수 * 건별절감시간 * 작업일

      3) 월별 목표 절감시간 : 일별목표건수 * 건별절감시간 * 작업일

    • 수행건수 : 작업수행 기록에서 상태가 PASS인 건만 카운트
    • 수행절감시간 : 수행건수 * 건별절감시간
    • 달성율-시간 : 수행절감시간 / 목표절감시간 * 100

정상적으로 로그아웃 되었습니다.
